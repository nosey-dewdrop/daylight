# Paper Doll Drawing Guide

## Canvas
- Draw at **384x384** (8x zoom of 48x48)
- Each "pixel" = 8x8 block in Procreate
- Export final as **48x48 PNG** (Nearest Neighbor resampling!)

## Templates (in /templates folder)
| File | Use |
|------|-----|
| TEMPLATE_hair_8x.png | Hair drawing guide (green=start, red=face NO-DRAW, blue=head edge) |
| TEMPLATE_tops_8x.png | Tops/shirts guide (green=torso zone, red=head NO-DRAW) |
| TEMPLATE_bottoms_8x.png | Pants/skirts guide (green=legs zone) |
| TEMPLATE_body_reference_8x.png | Clean body reference (no guides) |

## Coordinate Map (48x48 grid)
```
HEAD:    x:16-38, y:10-24  (center x:27)
FACE:    x:20-34, y:13-23  ← DON'T DRAW HERE for hair
EYES:    y:20-21
NECK:    y:25-27
TORSO:   y:28-38
LEGS:    y:39-46
CENTER:  x:27
```

## How to Draw
1. Open 384x384 canvas in Procreate
2. Import template as bottom layer (lock it)
3. Draw your part on a NEW layer above
4. Hide template layer
5. Export ONLY your drawing layer as 48x48 PNG

## Color Rules
- Outline: always #191412 (dark brown-black)
- Use 2-3 tones per part (base + highlight + shadow)
- Keep transparent background

## File Naming
- Hair: `hair_NAME.png` (e.g. hair_bob.png, hair_messy_bun.png)
- Tops: `top_NAME.png` (e.g. top_hoodie.png, top_crop.png)
- Bottoms: `bottom_NAME.png` (e.g. bottom_jeans.png, bottom_skirt.png)
