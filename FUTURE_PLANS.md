# daylight — future plans

## current state
baby blue typewriter aesthetic, TabView with Letters/Write/Profile tabs, envelope-style letter cards, compose view with ruled paper, avatar picker (placeholder), letter detail view.

## next up
- [ ] custom drawn stamps (pixel art, perforated edges) — replacing emoji placeholders
- [ ] custom drawn avatar parts (skin, eyes, mouth, hair) — paper doll layered PNGs
- [ ] envelope pixel art asset as letter card background
- [ ] stamp store — unlock stamps with XP, premium stamps
- [ ] avatar builder — full customization with accessory unlocks

## big features
- [ ] habbo-style pixel room / interior shop
  - users decorate their own room
  - invite friends to visit
  - furniture & decor items unlockable with XP or in-app
- [ ] bottle mail — throw a letter into the ocean, random stranger receives it
- [ ] memory box — letters you write but never send (private journal)
- [ ] time capsule letters — schedule delivery for a future date
- [ ] explore / pen pal discovery — find people to write to
- [ ] onboarding flow — register, pick avatar, set interests
- [ ] supabase auth + real-time letter delivery
- [ ] push notifications — "a letter has arrived"
- [ ] distance-based delivery time (haversine formula already implemented)
- [ ] music/song attachment to letters

## design direction
- pixel art + typewriter aesthetic
- baby blue palette, warm parchment paper textures
- nostalgic, slow, intentional — "real things take time"
- inspired by forget-me-not (web version: github.com/nosey-dewdrop/forget-me-not)

## i am open to any kind of recommendations!
if you have ideas for features, design improvements, or anything — feel free to suggest.

## tech stack
- SwiftUI (iOS 17+)
- Supabase (auth, database, realtime)
- custom pixel art assets (Procreate)
